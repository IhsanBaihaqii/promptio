# Promptio

Static premium landing app for Promptio with solid dark/neon styling, routed pages, a login gate for `/prompt` and `/profile`, prompt search, copy actions, and image protection hooks.

## Run locally

```powershell
node server.js
```

Then open:

```text
http://localhost:4173/home
```

Demo login:

```text
creator@promptio.id
promptio
```
